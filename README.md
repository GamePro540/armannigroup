## 📖 Overview
This repository hosts the official front-end architecture for the **Armanni Group** portfolio. Designed with a focus on minimalism, typography, and performance, this static site serves as the central hub for our active digital products, upcoming platforms, and development services.

## 🚀 Featured Projects
The portfolio highlights our core products and development branches:
* **Pinly:** A location-based iOS reminder utility built for simplicity and reliability.
* **GD Studio:** Our dedicated web architecture and digital design branch.
* **YUPP (In Development):** A Flutter-based, video-first live auction and marketplace application featuring real-time bidding (LiveKit) and Stripe integration.
* **Dejt (In Development):** A SwiftUI dating platform engineered to prioritize authentic conversation over immediate visual judgments.

## 🛠️ Technical Stack
This landing page is built to be extremely lightweight and fast, relying on native web technologies without the overhead of heavy frameworks:
* **HTML5:** Semantic structuring for accessibility and SEO.
* **Vanilla CSS3:** Utilizing CSS Grid, Flexbox, and custom CSS variables for a responsive, component-based design.
* **JavaScript:** Minimal, dependency-free scripting for smooth scrolling and basic email obfuscation/bot protection.
* **Hosting:** Deployed globally via GitHub Pages with custom DNS routing.
